## 🔁 Core Idea: Nested Loops

Most pattern problems use **nested loops**:

```text
Outer Loop → Rows
Inner Loop → Columns (stars/numbers/characters)
```

Example:

```cpp
for(int i = 1; i <= n; i++)        // rows
{
    for(int j = 1; j <= i; j++)    // columns
    {
        cout << "*";
    }
    cout << endl;
}
```
